Informações sobre versionamento de código

O que é versionamento na programação?

Na informática, o versionamento de software é o processo de atribuir um nome único ou uma númeração única para indicar o estado de um programa de computador. Esses números são geralmente atribuídos de forma crescente e indicam o desenvolvimento de melhorias ou correção de falhas no software.

O que é GitHub?

O GitHub é um serviço de hospedagem de código-fonte baseado na web que oferece controle de versão e recursos de colaboração para desenvolvedores. Fundado em 2008, rapidamente se tornou uma ferramenta indispensável no mundo da tecnologia.
